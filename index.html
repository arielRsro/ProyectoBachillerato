Bachillerato técnico

El bachillerato técnico especializado en Informática se enfoca en aprender como funcionan los aparatos que nos rodean, enfocado especialmente en computadoras y lo que pueden hacer.

¿Qué contiene esta especialidad?

En general puedes intuir las materias que contiene, como por ejemplo Programación o Soporte Técnico, pero también verás cosas como Aplicaciones ofimáticas y Sistemas operativos entre otras. En este sitio web podrás ver un poco de lo que trata Programación y Desarrollo web.

Programación

Por medio de la programación se diseña, codifica, limpia y protege el código fuente de los programas de los ordenadores, aprender de ella es indispensable, para resolver los desafíos en los sectores que lo requieran Unos de los beneficios al saber programar profesionalmente serían : Amplio abanico de oportunidades laborales. Una gran capacidad de análisis y de organización. Mayor capacidad para la solución de problemas de forma metódica. Posibilidades de tener tu propia empresa. Mayor autonomía profesional La programación es esencial para comprender el mundo que nos rodea Elaborar programas para la resolución de problemas mediante ordenadores.

Diseño Web

El concepto de página web se puede resumir en: un documento adaptado a la World Wide Web que puede tener información en forma de texto, imágenes, videos, música, entre otros tipos de multimedia. Igual se le conoce como página electrónica o web page en inglés.

¿Y qué es la Web?

La Web, World Wide Web o www, es donde todos los sitios web del mundo se entrelazan entre sí, y a la que podemos acceder mediante internet. Hay muchos tipos de sitios web, por ejemplo: un blog, landing page, wiki, sitio de descargas, sitio de subastas, de noticias, institucional, de comunidades, con bases de datos, de política, de educación, entre otros. Por eso es que tener un sitio web bien estructurado, optimizado y que tenga un buen diseño web profesional te asegura captar la atención de tus posibles clientes.
